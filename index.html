flash games instructions

  1. Find any flash game .swf file you want on archives such as this (GitHub).
  2. Download the file.
  3. Open an online .swf player. I have created a very simple but functional one that can be accessed href="https://m3tr0m0l4.github.io/flashplayer/">here. Alternatively, This one works and is not blocked.
  4. Upload your downloaded .swf file.
  5. Fullscreen and enjoy!
