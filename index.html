flash games instructions

  1. Find any flash game .swf file you want on archives such as this (GitHub).
  2. Download the file.
  3. Open an online .swf player. This one works and is not blocked.
  4. Upload your downloaded .swf file (on the website linked, there is an "Open File" button if you scroll down a bit and look next to the purple "Play" button).
  5. Click Play and Fullscreen!
